So what the hell is this?

This site exists so that you can find out how it was made, and so that I can too. While I am making it.

I will bump my head many times. But I'm okay with that. Our generation wants to learn , understand and - beyond everything - HAVE things immedaitely. This site is aimed and getting something up and running as soon as possible so that you have a playground to start building sand castles in.

Among other things, we will discuss:

  • Python for web
  • Flask as a framework
  • Bootstrap and (unfortunately) HTML - lazy version though
  • Deployment - again.. the lazy version.

No hand holding...

This is all about quick and easy right? So go ahead and download the latest Anaconda's package. You'll get Python interpreter and Spyder which is a great IDE to start with.

At the moment I'm using Notepad++ and running my programs in windows powershell. I've played with several IDE's and you end up staying with whatever you like and it doesn't make a difference. Spyder is a good one to start with though.
